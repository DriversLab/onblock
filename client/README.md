## OnBlock
OnBlock - it's interactive onboarding platform for Ton Ecosystem

## How to run

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Next run localtunnel or ngrok:
```bash
npx lt --port 3000
# or
ngrok http 3000
```

Copy that link and put to your bot settings.

## Do not forget about ENV
